# ⚠️ Rules While Solving

For EACH problem:

* 5 min think
* 2 min validate edge cases
* Code
* If WA → 3 min structured debug
* If still unclear → mark and move

After finishing each one, log:

* Time to idea
* Correct idea? (Y/N)
* WA count
* Why it failed (1 line)

No excuses.

## Key takeways

